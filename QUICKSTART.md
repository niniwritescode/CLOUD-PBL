# 🚀 Quick Start Guide

## Get up and running in 3 minutes!

### 1️⃣ Install Dependencies
```bash
npm install
```

### 2️⃣ Make sure MongoDB is running
**Windows:**
```bash
mongod
```

**macOS:**
```bash
brew services start mongodb-community
```

### 3️⃣ Seed the database
```bash
npm run seed
```

### 4️⃣ Start the server
```bash
npm start
```

### 5️⃣ Open in browser
```
http://localhost:5000
```

That's it! 🎉
